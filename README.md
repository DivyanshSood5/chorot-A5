# Øliv — Brand Agency Landing Page

A pixel-faithful clone of the **Øliv** brand agency website, built with pure HTML and CSS.

## Preview

A clean, modern landing page for a brand agency featuring:

- Responsive navigation with hiring badge
- Animated logo cloud marquee
- Services & selected work showcase
- Our values grid layout
- Team members section
- Customer testimonials
- Blog / insights cards
- Full footer

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styles | CSS3 (Flexbox, Grid, custom animations) |
| Icons | [Remix Icon](https://remixicon.com/) |
| Fonts | [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts |

## Getting Started

No build step needed — just open in a browser:

```bash
open index.html
```

Or serve locally:

```bash
npx serve .
```
